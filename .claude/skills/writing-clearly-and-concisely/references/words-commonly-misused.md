# Words and Expressions Commonly Misused

Quick reference for common word choice issues.

## Word Pairs

| Wrong | Right | Note |
|-------|-------|------|
| affect | effect | affect=verb, effect=noun (usually) |
| aggravate | irritate | aggravate=worsen, irritate=annoy |
| among | between | among=3+, between=2 |
| amount | number | amount=mass, number=countable |
| compare to | compare with | to=similarity, with=differences |
| comprise | compose | whole comprises parts; parts compose whole |
| continual | continuous | continual=recurring, continuous=unbroken |
| disinterested | uninterested | disinterested=impartial, uninterested=bored |
| farther | further | farther=distance, further=degree |
| fewer | less | fewer=countable, less=mass |
| imply | infer | writer implies, reader infers |
| its | it's | its=possessive, it's=it is |
| lay | lie | lay takes object, lie does not |
| literally | figuratively | literally=actually happened |
| principal | principle | principal=main/person, principle=rule |
| that | which | that=restrictive, which=non-restrictive |
| who | whom | who=subject, whom=object |

## Quick Tests

**"The fact that" test:** Can you rephrase without it? Do so.
```
❌ I was unaware of the fact that the deadline had passed.
✅ I was unaware the deadline had passed.
```

**"Very" test:** Delete "very" and choose a stronger word.
```
❌ very tired → ✅ exhausted
❌ very happy → ✅ elated, thrilled
❌ very important → ✅ crucial, essential
```

**Passive test:** Can you name the actor? Make them the subject.
```
❌ Mistakes were made.
✅ The team made mistakes.
```
